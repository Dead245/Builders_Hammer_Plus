# Builder's Hammer+

Mod for Hytale.

Adds more functionality to the vanilla Builder's Hammers.  Also adds 1 new hammer, the the Thorium Builder's Hammer.

<img width="582" height="373" alt="HammerDescription" src="https://github.com/user-attachments/assets/136c9006-b1ed-48ad-997a-0fd0d9389a6a" />

All Builder's Hammers now have multiple functions:
1. Left Click -  Cycle Block Set (Vanilla behavior)
2. Shift + Left Click - Cycle Block State
3. Right Click - Rotate Block on Y axis
4. Shift + Right Click - Rotate Block on X/Z axis
5. 'Select Block From Set' keybind while holding a block: Swap to a different block in that block's set. (vanilla behavior)
